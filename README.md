## How to Install
1. Download Latest ```.sql``` file release.
2. Create database ```tugas-akhir```.
2. Import database to phpmyadmin.
3.  Run in terminal: ```composer update``` or ```composer install```.
4.  Run ````php artisan serve````.

## How to Install with migration (not implemented)
1. Create database ```tugas-akhir```.
2. Run ```php artisan migrate --seed``` or to refresh ```php artisan migrate:refresh --seed```
3.  Run in terminal: ```composer update``` or ```composer install```.
4.  Run ````php artisan serve````.
